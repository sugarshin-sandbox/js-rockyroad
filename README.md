## Getting started

```
git clone git@github.com:sugarshin/js-rockyroad.git
cd js-rockyroad
npm install
```

## Usage

```
# テスト
npm test -- test/filename.js

# 全テスト
npm run test:all

# ファイル変更監視
npm run tdd
```
